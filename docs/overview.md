# Overview

The micropayments system is the consumption layer of a DAO-governed treasury ecosystem on Stellar.

## Ecosystem Roles

- **Treasury contract**: governance, proposal approval, and fund allocation
- **Micropay contract**: user-level balance accounting and request charging
- **Backend service**: API key auth + transaction orchestration
- **Frontend dashboard**: developer visibility into key, balance, and usage

## Mission

Enable verifiable pay-per-use API access where treasury allocations are consumed transparently and auditable on-chain.
